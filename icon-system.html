Icon Design Principles

Consistency Rules

  • ✓ 2px stroke width for all outline icons
  • ✓ Rounded line caps and joins for friendly appearance
  • ✓ 24x24px base size with 2px padding
  • ✓ Consistent corner radius: 4px for squares, 12px for rounded rectangles
  • ✓ Primary purple (#a855f7) for main actions
  • ✓ Secondary blue (#3b82f6) for supporting elements
  • ✓ Gradient for special/premium features

Implementation Guide

How to Use Icons in HTML

<!-- Inline SVG with classes --> <svg class="tw-icon tw-icon-primary" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2"> <path d="M3 9l9-7 9 7v11a2 2 0 0 1-2 2H5a2 2 0 0 1-2-2z"/> </svg> <!-- Or as a reusable symbol --> <svg class="tw-icon tw-icon-lg tw-icon-gradient"> <use href="#icon-couple-hearts"></use> </svg>

Icon Sizes

  • tw-icon-sm - 16x16px (inline text)
  • tw-icon - 24x24px (default)
  • tw-icon-lg - 48x48px (feature icons)

Icon Colors

  • tw-icon-primary - Purple (#a855f7)
  • tw-icon-secondary - Blue (#3b82f6)
  • tw-icon-success - Green (#22c55e)
  • tw-icon-warning - Orange (#f59e0b)
  • tw-icon-danger - Red (#ef4444)
  • tw-icon-gradient - Purple to Blue gradient
